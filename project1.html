South Asian Clothing
CNN Classifier

Client:

DS340

Role:

Co-Lead

Year:

2024

See Full Report

This project focuses on the development of an advanced deep learning model designed to classify South Asian fashion items, such as sarees, kurtas, lehengas, and sherwanis. By leveraging a custom Convolutional Neural Network (CNN) architecture, the model effectively captures intricate design patterns, textures, and styles unique to South Asian attire. The project addresses common challenges in fashion classification, including dataset imbalance, feature extraction for diverse fabric patterns, and variations in traditional clothing silhouettes.

Objective:

The primary objective of this project was to design and implement a robust CNN architecture capable of accurately classifying diverse South Asian fashion items across multiple categories. The model aimed to improve classification performance by addressing class imbalance, optimizing feature extraction, and enhancing generalization. Additionally, the project sought to explore the scalability of the model for real-world applications in fashion e-commerce platforms, cultural heritage archiving, and automated tagging systems for large-scale fashion databases.

Approach:

The project utilized a comprehensive dataset of over 100,000 high-resolution fashion images, systematically divided into training (70%), validation (15%), and test (15%) sets to ensure robust performance evaluation. Key methodologies included:

  • Data Preprocessing: Implementation of advanced data augmentation techniques such as image resizing, horizontal and vertical flipping, rotation, zooming, and color jittering to enhance model generalization and reduce overfitting.
  • Class Balancing: Addressed class imbalance through the application of weighted cross-entropy loss functions and the integration of a WeightedRandomSampler to ensure fair representation of all fashion categories, particularly underrepresented classes.
  • Custom CNN Architecture: Designed a multi-layered CNN with stacked convolutional layers, batch normalization, ReLU activations, max-pooling layers, and dropout regularization. This architecture was optimized to capture both low-level features (textures, patterns) and high-level semantic features (shapes, cultural motifs).
  • Training and Optimization: Utilized the Adam optimizer for efficient gradient descent, with a learning rate scheduler to dynamically adjust learning rates during training. The model was trained using a categorical cross-entropy loss function, achieving an accuracy of up to 85% on the test set.
  • Evaluation Metrics: Performance was assessed using confusion matrices, precision-recall curves, and F1-scores to analyze the model’s classification capability across different categories. The model demonstrated strong generalization, particularly in distinguishing visually similar garments.

Impact & Future Work:

This AI-driven solution demonstrates significant potential for real-world applications, including automated fashion tagging for e-commerce platforms, cultural fashion archiving, and personalized recommendation systems. Future work will focus on integrating transfer learning techniques using pre-trained models like ResNet and EfficientNet to further improve accuracy. Additionally, incorporating attention mechanisms and exploring multi-label classification approaches will enhance the model’s ability to recognize overlapping categories and complex garment attributes.
