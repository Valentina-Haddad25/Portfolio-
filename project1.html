South Asian Clothing
CNN Classifier

Client:

DS340

Role:

Co-Lead

Year:

2024

See Full Report

This project develops an efficient deep learning model to classify South Asian fashion items, such as sarees, kurtas, and lehengas, into distinct categories. By leveraging a custom convolutional neural network (CNN) and data balancing techniques, the model enhances classification accuracy while addressing dataset imbalances.

Objective:

To design a robust CNN architecture capable of accurately classifying diverse fashion items while ensuring fairness across categories through class-balancing methods. The goal is to create a scalable and adaptable model suitable for real-world applications in e-commerce and fashion analytics.

Approach:

The project utilized a dataset of over 100,000 fashion images, systematically split into training, validation, and test sets. To improve model generalization, data augmentation techniques such as resizing, rotation, and color jitter were applied. A custom CNN architecture was designed with multiple convolutional layers, ReLU activations, and dropout regularization to optimize feature extraction. Additionally, class-balancing techniques such as weighted cross-entropy loss and a WeightedRandomSampler ensured fair representation of all fashion categories. The model was trained using the Adam optimizer, achieving up to 85% classification accuracy.
